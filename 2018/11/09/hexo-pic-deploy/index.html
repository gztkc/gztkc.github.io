Hexo 3.x 版本博客发布图片显示问题解决

发表于 2018-11-09 | 分类于 Hexo

环境说明

  • hexo版本 3.8.0

  • pages服务供应商 码云（https://gitee.com/）

hexo默认是无法处理插入的本地图片，但通过扩展插件则能实现本地图片插入。

配置和插件安装

  1. 设置post_asset_folder值
    配置hexo根目录下_config.yml里面的post_asset_folder:false属性为true

  2. 安装扩展插件hexo-asset-image
    hexo根目录下执行npm install hexo-asset-image --save

安装完该插件后，使用hexo n "xxxx"生成新博客时候，/source/_posts文件夹内除了xxx.md文件外还会生成一个与之对应的同名文件夹。

  1. 引入图片
    在xxx.md中引入图片的时候，先把图片复制到对应的文件夹中，然后使用markdown语法引入图片即可
    ![desc](xxx/pic.jpg)

图片显示问题

按照上述操作进行了图片引入后发现，不管是hexo s本地启动进行页面查看，还是hexo g -d发布后进行页面查看，图片也无法如期显示，如下图：

使用浏览器捕获一下显示元素
